## Como compilar?

### No VSCode

Importe um projeto Raspberry Pi Pico, com a versão Pico SDK 2.1.0. É necessário ter instalado a extensões Raspberry Pi Pico.

### Manualmente

- > mkdir build/

- > cd build

- > cmake ..

- > make
